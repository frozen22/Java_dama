Jedna se o implementaci hry Dama v programovacim jazyce Java SE 7. Projekt byl vytvoren pro predmet Seminar Java (IJA) v akademimickem roce 2012/2013 FIT VUT.
Aplikace vyuziva grafickeho rozhrani pro interakci s uzivatelem.

Do zakladni funkcnosti patri hry: hrac proti UI, hrac proti hraci(Local game),hrac proti vzdalenemu hraci(Online game). Dale umi prehravat ulozene zaznamy her ve formatu
xml i v zakladni notaci.
